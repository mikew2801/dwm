* Remap caps lock

#+begin_src bash :tangle void_setup.sh :results silent
# Now enable xmodmap
cat << EOF > ~/modmap
       !
       ! Swap Caps_Lock and Control_L
       !
       remove Lock = Caps_Lock
       remove Control = Control_L
       keysym Control_L = Caps_Lock
       keysym Caps_Lock = Control_L
       add Lock = Caps_Lock
       add Control = Control_L
EOF

echo "Content has been written to ~/modmap."
#+end_src

* Xresources

You can make EXWM/DWM (or any window manager) viewable by this. You might want to use different settings for laptop/desktop mode.

Best for laptop screen: *144* (Xft.dpi); best for HDMI output: *126*.

#+begin_src config :tangle ~/.Xresources
Xft.dpi: 144
Xcursor.theme: Breeze-Adapta-Cursor
Xcursor.size: 24
Xft.antialias: 1
Xft.hinting: 1
Xft.hintstyle: hintslight
Xft.rgba: rgb
#+end_src

** Toggle touchpad (Super+Shift+t)

Simple script to toggle the touchpad:

#+begin_src shell :tangle ~/scripts/tt.sh
#!/bin/bash
# toggle_touchpad.sh
# Toggles the touchpad on/off using xinput

# Replace this with your device ID or exact name
DEVICE="10"  # or "SynPS/2 Synaptics TouchPad"

# Get current enabled status (0 = disabled, 1 = enabled)
STATUS=$(xinput list-props "$DEVICE" | awk '/Device Enabled/{print $4}')

if [ "$STATUS" -eq 1 ]; then
    xinput disable "$DEVICE"
    echo "Touchpad disabled."
else
    xinput enable "$DEVICE"
    echo "Touchpad enabled."
fi
#+end_src
